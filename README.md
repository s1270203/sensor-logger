# センサーデータロガー（C言語）

このプロジェクトは、C言語で実装されたシンプルなセンサーデータ処理プログラムです。  
センサーデータの「平均」「最大・最小」「外れ値除去」「CSV形式での整形出力」を行います。

---

## 主な機能

- センサーデータ（構造体）から統計値を計算
- 外れ値を除外した平均の計算
- データをCSV形式で出力
- `snprintf()`による安全な文字列整形

---

## ビルド方法

```bash
make
./sensor_logger
```

## クリーンアップ

以下のコマンドで、コンパイル時に生成された中間ファイル（`.o`ファイルや実行ファイル）を削除できます：

```bash
make clean
```
